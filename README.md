# GamerBot
[![works badge](https://cdn.rawgit.com/nikku/works-on-my-machine/v0.2.0/badge.svg)](https://github.com/nikku/works-on-my-machine)

겜스터 디스코드 봇입니다.

겜스터 멤버들이 각 컨텐츠별로 내전, 게임별로 플레이시, 유틸리티 디스코드 봇입니다.
